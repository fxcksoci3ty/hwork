1. Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?

ОТВЕТ: Чтобы склонировать на локальный компьютер репозиторий из github.com нужно воспользоваться командой git clone и в качестве параметра указать SSH URL по типу git@github.com:user/repo.git. Предварительно должен быть добавлен SSH ключ в github аккаунт.

2. Как создать git репозиторий из локального каталога на своем компьютере. Как его отправить на github.com?

ОТВЕТ: Находясь в каталоге, нужно проинициализировать пустой git репозиторий, с помощью команды git init. Далее, чтобы связать его с созданным репозиторием на github.com нужно выполнить команду git remote add origin и указать URL репозитория, как делали в первом вопросе (git@github.com:user/repo.git).

3. Как получить на локальный компьютер изменения с github.com?

ОТВЕТ: За это отвечает команда git pull.

4. Какую информацию нам дают команды:
	git status
	git log
	git remote -v 

ОТВЕТ: git status показывает состояние репозитория, файлы, которые были изменены, добавлены, удалены, но ещё не закоммичены. Тут также отображается текущая ветка, в которой ведётся работа. git log это вывод истории коммитов в репозитории, а также информации о каждом коммите (хэш, автор, дата и время + сообщение коммита). git remote -v показывает список удалённых репозиториев, связанных с локальным репозиторием. -v показывает URL и имена удалённых репозиториев.
